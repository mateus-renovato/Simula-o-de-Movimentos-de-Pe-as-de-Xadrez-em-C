♟️ Simulação de Movimentos de Peças de Xadrez em C

Este projeto foi desenvolvido em linguagem C para simular os movimentos de algumas peças do xadrez utilizando conceitos fundamentais de programação.

O programa demonstra:

• Recursividade para movimentação da Torre, Bispo e Rainha
• Estruturas de repetição (loops) para o movimento do Cavalo
• Organização do código em funções
• Simulação textual dos movimentos no terminal

O objetivo do projeto é praticar lógica de programação, controle de fluxo e organização de código em C.

Movimentos implementados:
- Torre: movimento horizontal
- Bispo: movimento diagonal
- Rainha: movimento horizontal
- Cavalo: movimento em L (2 casas para cima e 1 para direita)

Tecnologias utilizadas:
- Linguagem C
- Compilador GCC
